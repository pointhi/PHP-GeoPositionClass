PHP-GeoPositionClass

A simple PHP-Class which convert different Geographical Coordinate Systems

Overview

GeoPosition.class.php is a PHP Class to convert Geo-Data like Coordinates. The Data-Input-Handling is unlike other programs very fault tolerant.

The Class can handle Coordinates in this formats:

  • Locator System
  • Coordinate Decimal
  • Coordinate Sexagesimal

Working with the Class

  1. As first you create an Object from Type "GeoPosition"
  2. Then you can give the Class Coordinates with the Set-Functions
  3. Now You can get the Coordinates in a new Format with the Get-Functions
